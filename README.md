# Jinyu Cai

PhD Student, Waseda University

I am a PhD student at **Waseda University**, working on the intersection of **Large Language Models (LLMs), Software Engineering, and Autonomous Systems**.

My research focuses on applying **LLMs to improve software reliability and intelligent system development**, including:

- LLM-based software testing
- AI-assisted program analysis
- multi-agent simulation with LLMs
- evolutionary computation with LLM guidance
- autonomous planning evaluation in complex environments
- AI-driven testing for games and complex systems
