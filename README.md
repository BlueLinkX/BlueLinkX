## Profile

I am a Ph.D. student at **Waseda University**, working at the intersection of **Large Language Models (LLMs), Software Engineering, and Autonomous Systems**. My recent research focuses on how LLMs can be used to improve software reliability, automate testing and analysis, and support the development and evaluation of intelligent systems.

My current research can be summarized into four connected themes:

- **LLM-based software testing and game quality assurance**: automated smoke testing, regression testing, incremental playtesting, and benchmark design for complex game and interactive environments.
- **LLM-assisted program analysis and self-adaptive systems**: requirement analysis, adaptation-rule generation and optimization, and semantics-aware controller synthesis.
- **LLM-based multi-agent simulation and computational social systems**: simulation of language evolution, regulated communication, and agent behavior under constrained environments.
- **Computational optimization and sequential decision-making**: value iteration architectures, deep reinforcement learning, and evolutionary algorithms for planning, control, and evaluation in autonomous and interactive systems.

## Research Experience Highlights

### 1. LLM-based Software Testing and Game Playtesting

- Developed LLM-driven methods for automated game smoke testing, regression testing, and incremental playtesting in dynamic game environments.
- Built evaluation assets and testbeds for complex interactive systems, including multi-ending quests and cooking-task planning scenarios.
- Representative outputs include work on automated game smoke testing, **SAGE** for gray-box regression testing, knowledge-graph-enhanced incremental playtesting, and **Overcooked Plus** as a planning evaluation testbed.

### 2. LLMs for Self-Adaptive Systems and Program Analysis

- Studied how LLMs can support requirement analysis, adaptation-rule generation, adaptation-rule optimization, and controller synthesis.
- Focused on reducing manual effort in the design and analysis of adaptive software systems while preserving semantic interpretability.

### 3. LLM-based Multi-Agent Simulation and Computational Social Systems

- Built LLM-based multi-agent simulation frameworks to study how language strategies evolve under platform regulation and constrained communication settings.
- Combined LLMs with evolutionary or optimization components to model social adaptation and regulation evasion.

### 4. Optimization, Reinforcement Learning, and Autonomous-System Evaluation

- Investigated value-iteration architectures, deep reinforcement learning, and genetic-algorithm-based optimization for intelligent decision making.
- Studied how language models can assist optimization workflows, especially in evolutionary computation settings.
- Evaluated VLM/LLM capabilities in autonomous-driving settings and studied planning and safety evaluation in embodied and autonomous systems.
