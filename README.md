## About Me / 自己紹介

I am a Ph.D. student at Waseda University working at the intersection of large language models, software engineering, and autonomous systems. My research focuses on software reliability, automated testing and analysis, and the development and evaluation of intelligent systems.

早稲田大学博士後期課程に在籍し、LLM、ソフトウェア工学、自律システムの交差領域で研究を行っています。主な関心は、ソフトウェアの信頼性向上、テスト・解析の自動化、知能システムの開発と評価です。

---

## Research Interests / 研究分野

- **LLM-based software testing and game QA / LLMを用いたソフトウェアテスト・ゲームQA**: automated smoke testing, regression testing, incremental playtesting, and benchmark design for complex game and interactive environments.
- **LLM-assisted program analysis and self-adaptive systems / LLMを活用したプログラム解析・自己適応システム**: requirement analysis, adaptation-rule generation and optimization, and semantics-aware controller synthesis.
- **LLM-based multi-agent simulation and computational social systems / LLMベースのマルチエージェントシミュレーション**: language evolution, regulated communication, and agent behavior under constrained environments.
- **Optimization and sequential decision-making / 最適化・逐次意思決定**: value iteration architectures, deep reinforcement learning, and evolutionary algorithms for planning, control, and evaluation.
